Git

  • Git is a version control system that allows developers to manage and keep track of each version of a project.
  • Git commands start with "git"
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pull down the most recent main branch code to the local machine
  • git add -A: adds all changes from the local branch to the staging area
  • git push origin feature/branch-name: pushes the changes made locally to the working branch in GitHub
  • Once changes are pushed to GitHub, a PR can be made to merge them back to the main branch.
